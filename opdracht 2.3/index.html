Opdracht 2.3: Onderzoeken

Waarom noem je de eerste HTML-pagina; index.html?

De naam "index.html" is een conventie die door webservers wordt gebruikt. Wanneer je een domeinnaam zonder een specifieke bestandsnaam invoert, zoals www.example.com, zoekt de server standaard naar een bestand met de naam "index.html" om weer te geven. Dit komt doordat de "index.html" meestal de hoofdpagina van een website is.

Met welke HTML versie werken we nu?

We werken momenteel met HTML5

Met welke CSS versie werken we nu?

De huidige versie van CSS is CSS3. CSS3 is niet één enkele versie, maar een verzameling modules die afzonderlijk zijn ontwikkeld.

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Verschillende browsers gebruiken verschillende rendering engines om HTML en CSS te interpreteren en weer te geven.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

De populairste browser is momenteel Google Chrome, met een marktaandeel van ongeveer 65-70% wereldwijd.Het exacte percentage RAM dat Chrome gebruikt, varieert per systeem en situatie, maar het kan makkelijk 10-50% van het beschikbare RAM innemen.
